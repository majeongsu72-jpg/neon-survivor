# 🌟 NEON SURVIVOR — 네온 서바이버

쏟아지는 네온 괴물 무리에서 **10분간 살아남는** 브라우저 생존 액션 게임입니다.
뱀파이어 서바이버 스타일 — 이동만 하면 공격은 자동, 레벨업마다 카드를 골라 나만의 빌드를 완성하세요.

### ▶️ [지금 플레이하기](https://majeongsu72-jpg.github.io/neon-survivor/)

설치 불필요 · PC/모바일 지원 · 단일 HTML 파일 (외부 라이브러리, 이미지/사운드 에셋 없음)

## 🎮 조작

| 키 | 동작 |
|---|---|
| `WASD` / 방향키 | 이동 (공격은 자동!) |
| `P` / `ESC` | 일시정지 |
| `M` | 소리 켜기/끄기 |
| `1` `2` `3` | 레벨업 카드 선택 |
| 모바일 | 화면 드래그 = 가상 조이스틱 |

## ⚔️ 게임 요소

- **무기 6종** (최대 4개, 각 5레벨): 매직 볼트 🔮 · 회전 수리검 🌀 · 천둥 번개 ⚡ · 화염 파동 🔥 · 유도 미사일 🚀 · 프리즘 레이저 💠
- **패시브 6종**: 파워 코어 · 오버클럭 · 네온 부츠 · 강화 심장 · 자석 장갑 · 나노 재생
- **적 6종 + 보스 3종** (3분/6분/9분) + 엘리트 몬스터 (처치 시 보물상자)
- **드롭 아이템**: 치킨 🍗 (회복) · 자석 🧲 (보석 전체 흡수) · 폭탄 💣 (화면 클리어)
- 데미지 숫자, 크리티컬, 콤보, 화면 흔들림, 히트스톱, 신기록 저장

## 🛠️ 기술

순수 바닐라 JavaScript + Canvas 2D 한 파일로 작성됐습니다.

- **렌더링**: 글로우 스프라이트 사전 렌더링 + additive blending (적 300마리에서 60 FPS)
- **사운드**: 효과음과 배경음악 모두 Web Audio API 실시간 합성 (오디오 파일 0개)
- **충돌**: 공간 해시 그리드
- 로컬에서 실행하려면 `index.html`을 브라우저로 열기만 하면 됩니다
